# rle-cpp

[![CI](https://github.com/5etrovich/rle-cpp/actions/workflows/ci.yml/badge.svg)](https://github.com/5etrovich/rle-cpp/actions/workflows/ci.yml)

Консольная утилита для сжатия и разжатия файлов алгоритмом RLE (Run-Length Encoding).

## Что такое RLE

Run-Length Encoding — простой алгоритм сжатия без потерь. Повторяющиеся последовательности байт заменяются парой `(количество, байт)`. Хорошо работает на данных с длинными серями одинаковых байт: BMP-изображения, простые текстуры, бинарные файлы с паддингом.

## Сборка

```bash
cmake -B build
cmake --build build
```

## Использование

```
./rle
```

Программа спросит, что нужно сделать:
- `1` — сжать файл (создаёт `<имя файла>.rle`)
- `2` — разжать файл (должен иметь расширение `.rle`)

Поддерживаются имена файлов с пробелами.
